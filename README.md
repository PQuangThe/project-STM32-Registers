# project-STM32-Registers
Đây là dự án lập trình stm32 bằng thanh ghi hạn chế dùng thư viện có sẵn
1. Lập trình các ngoại vi của stm32 trực tiếp bằng thanh ghi:
    - TIMER
    - USART
    - DMA
    - SPI
2. Tích hợp FreeRTOS
3. Đọc ghi bộ nhớ ngoài flash w25q16
